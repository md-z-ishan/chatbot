Covid Health Tracker 
The COVID-19 Health Tracker is a web-based application that allows users to monitor real-time COVID-19 statistics such as confirmed cases, recoveries, and deaths across different countries and regions. It is designed to provide clear visual insights through tables and interactive charts to help users understand the spread and impact of the virus.
